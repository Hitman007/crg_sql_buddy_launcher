== Description ==
Launches SQL Buddy directly from the Wordpress Dashboard, allowing direct access to the database. SQLBuddy is an open source data management application, similiar to PHPMyAdmin.

== Installation ==

Download the zipped file. Copy the directory inside this file to the “plugins” directory of your WordPress instillation. From the Plugins menu, click “activate plugin”.

== Frequently Asked Questions ==
This plugin launches SQLBuddy from the Wordpress dashboard. Please see the documentation for SQLBuddy for further information. Reference, http://sqlbuddy.com .
